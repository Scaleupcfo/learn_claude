# Connect GitHub

About 2 minutes. Optional — for CXOs who want a window into engineering velocity.

## Add the connector

1. Open [claude.ai/settings/connectors](https://claude.ai/settings/connectors).
2. Find **GitHub** and click **Connect**.
3. Authorize the orgs / repos you want Claude to see. **Be specific** — don't grant access to private repos blanket-style.

<Checkpoint id="connector.github.added">GitHub is connected.</Checkpoint>

## Try it now

```text
Look at the [ORG/REPO] repository over the last 30 days. Summarize: what shipped, who's most active, any open PRs that have been waiting more than 5 days, and a one-sentence health assessment.
```

```text
Open the README of [ORG/REPO]. Rewrite the first paragraph so a board member could understand what this software does.
```
