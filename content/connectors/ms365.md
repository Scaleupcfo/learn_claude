# Connect Microsoft 365

About 3 minutes. Covers Outlook, Word, Excel, PowerPoint, OneDrive, and Teams in one shot — for Office-first execs.

## Add the connector

1. Open [claude.ai/settings/connectors](https://claude.ai/settings/connectors).
2. Find **Microsoft 365** and click **Connect**.
3. Sign in with your Microsoft / work account. You may need your IT admin to approve org-wide access if you're on a managed tenant — show them this page if asked.

<Checkpoint id="connector.ms365.added">Microsoft 365 is connected.</Checkpoint>

## What Claude can do

- 📧 **Outlook** — read, draft, send (with confirmation), search calendar.
- 📄 **Word** — open, edit, create docs in OneDrive.
- 📊 **Excel** — open, analyze, generate workbooks.
- 🎞️ **PowerPoint** — create decks; restyle slides.
- 💬 **Teams** — read channel summaries, draft messages.

## Try it now

```text
Find my last 10 Outlook emails with [PERSON]. Summarize the relationship, the last thing they asked, and what I owe them.
```

```text
Open the PowerPoint at [URL] and rewrite the headers so each one is a complete sentence — the takeaway, not the topic.
```
